# Design System: 3M Clinic (Rolex & Coffee Concept)
*Premium Medical Editorial Style*

## 1. Цветовая палитра (Color Palette)

### Базовые цвета (Backgrounds)
- **Cotton Cream (`#F4EFE6`)**
  *Основной фон приложения.* Теплый кремовый оттенок с имитацией матовой хлопковой бумаги. Задает ощущение дорогой визитки и "старых денег".
- **Linen Surface (`#FCFAF6`)**
  *Фон карточек.* Чуть более светлый оттенок кремового (почти белый, но без синевы). Выделяет контентные блоки (услуги, быстрые действия) на фоне хлопковой бумаги, создавая мягкий контраст без грубых теней.
- **Warm Border (`#E8DED1`)**
  *Линии и разделители.* Цвет топленого молока. Используется для тончайших рамок в 1px. Заменяет серый цвет, делая интерфейс "бесшовным" и теплым.

### Текст (Typography Colors)
- **Deep Espresso (`#2A2624`)**
  *Основной текст и акценты.* Глубокий кофейно-графитовый цвет. Заменяет глухой черный (`#000000`). Не режет глаз, выглядит благородно на кремовом фоне.
- **Warm Muted (`#9E9892`)**
  *Второстепенный текст.* Приглушенный серо-бежевый. Используется для подзаголовков, неактивных иконок и надписей.

### Акценты (Brand Accents)
- **Rolex Green (`#006039`)**
  *Главный бренд-цвет.* Классический зеленый цвет циферблатов Rolex. Используется для главных заголовков, главных кнопок CTA (Call to Action) и активного состояния в нижнем меню.
- **Terracotta Coffee (`#9C5426`)**
  *Второстепенный акцент.* Цвет ржавчины / жженого кофе. Заменяет традиционное "золото". Используется для бейджей ("Активная запись", "Хит", "Акция"), иконок и тонировки фотографий.

## 2. Типографика (Typography)

Интерфейс построен на шрифтовой паре "Антиква + Гротеск" (характерно для Vogue, Cartier, Rolex).

### Акцентный шрифт (Антиква)
**Cormorant Garamond**
- Используется только для крупных заголовков (H1, H2, H3) и цен (баланс, стоимость услуги).
- Начертания: `Bold (700)`, `Medium (500)`.
- Специфика: Не используется курсив, только строгое прямое начертание для ощущения медицинской стабильности.

### Наборный шрифт (Гротеск)
**Montserrat**
- Используется для всего остального текста, кнопок, бейджей и навигации.
- Начертания: `Medium (500)`, `Light (300)`.
- **Стиль "Micro-caps":** Все кнопки, бейджи и навигация пишутся мелким кеглем (`9px - 10px`), КАПСОМ (`uppercase`) с огромным расстоянием между буквами (`tracking-[0.15em] - [0.2em]`). Это визуальный код luxury-брендов.

## 3. UI-Элементы и Формы

### Карточки и Кнопки
- **Полное отсутствие закруглений (Border Radius: 0).**
  Никаких "мыльных" углов (`rounded-xl` или `rounded-full` для кнопок). Все карточки услуг, баннеры и кнопки строго прямоугольные. Исключение — только нижнее меню.
- **Тонкие рамки (Hairlines).**
  Эффект карточек достигается не за счет теней (shadows), а за счет тонких рамок цвета `Warm Border` (`#E8DED1`).

### Изображения
- **Тонировка (Blend Mode).**
  Все фотографии врачей обесцвечиваются (`grayscale`) и покрываются легким слоем кофейного цвета (`#9C5426` на 10% прозрачности в режиме `multiply`). Это делает разнобойные фотографии врачей единым стилистическим блоком, который не спорит с зеленой кнопкой и кремовым фоном.
- **Логотип.**
  Используется оригинальный логотип, который накладывается на кремовый фон через `mix-blend-multiply`, что "съедает" белый квадратный фон картинки и делает его прозрачным.

### Иконки
- **Phosphor Icons (Light / Fill).**
  Используется самое тонкое начертание (`ph-light`) для всех интерфейсных иконок. 
  Для активного состояния (например, нижнее меню) используется залитое начертание (`ph-fill`) в цвете Rolex Green.
