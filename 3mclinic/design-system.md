# Дизайн-система: 3M Clinic

Собрана на основе анализа стилей сайта `https://3mclinic.ru/`

## 1. Цветовая палитра (Colors)

### 🟢 Фирменный (Brand Colors)
- **Primary Mint:** `#7ED6A5` (rgb: 126, 214, 165) — главный акцентный цвет (используется для иконок, заголовков, плашек).
- **Mint Hover:** `#56D38E` (rgb: 86, 211, 142) — цвет кнопок при наведении (более насыщенный).
- **Mint Dark:** `#3B8B60` — используется для контрастного текста на светлом фоне.
- **Mint Light 1:** `#F3F8F3` — очень светлая подложка под блоки (1 уровень).
- **Mint Light 2:** `#E9F2E9` — акцентная светло-мятная подложка (2 уровень, под иконки).

### ⚪️ Базовые (Base / Neutrals)
- **Background White:** `#FFFFFF` — основной фон карточек и секций.
- **Background Light Gray:** `#FAFAFA`, `#F2F2F7` — фон для разделителей и подложек.
- **Text Main (Black):** `#1B1B1B` (rgb: 27, 27, 27) — основной цвет текста, мягкий черный.
- **Text Muted (Gray):** `rgba(0, 0, 0, 0.5)` — вторичный текст, подписи.

---

## 2. Типографика (Typography)

Сайт использует микс нескольких шрифтов (подключены Google Fonts):

- **Заголовки (Headings):** `Inter` и `Montserrat`.
- **Основной текст (Body):** `Inter` и `Roboto`.
- **Начертания (Weights):** 
  - `300` (Light) — для длинного текста и описаний.
  - `500` (Medium) / `600` (Semi-bold) — для подзаголовков.
  - `700` (Bold) — для акцентных заголовков.

*Рекомендация для приложения:* Использовать `Inter` как единый системный шрифт для чистоты дизайна (он идеально подходит под Apple-стиль).

---

## 3. Элементы UI (Shape & Shadows)

### Скругления углов (Border Radius)
Сайт использует мягкие формы:
- **Малые элементы (кнопки, плашки):** `6px`, `10px`, `12px`.
- **Средние элементы (карточки врачей, прайса):** `20px`.
- **Крупные элементы (секции, фото):** `30px`.

### Тени (Box Shadows)
Тени используются для создания глубины, они мягкие и рассеянные:
- **Легкая (для карточек):** `0px 8px 20px 0px rgba(0, 0, 0, 0.08)`
- **Глубокая (для парящих элементов):** `0px 8px 32px 0px rgba(0, 0, 0, 0.3)`

---

## 4. Стилизация кнопок (Buttons)

- **Primary Button (Главная):** 
  - Фон: `#7ED6A5`
  - Текст: Белый
  - Скругление: `50px` (pill-shape, полностью круглые края)
- **Secondary / Ghost Button (Вторичная):** 
  - Фон: Прозрачный
  - Обводка: 1px `#7ED6A5`
  - Текст: `#7ED6A5`

---

## 5. Иконки
На сайте используются линейные минималистичные иконки с заливкой фирменным мятным цветом или с легкой мятной подложкой (`#E9F2E9`). 
Для приложения рекомендуется использовать библиотеку **Phosphor Icons** (стиль Fill для таббара и Light/Regular для контента) — они идеально совпадают по стилистике.
